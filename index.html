Vi text editor

starting vi

Open a file with vi. Type: vi myfile.txt
- If myfile.txt does not exist, a screen will appear with just a cursor at the top followed by tildes (~) in the first column.
-If myfile.txt does exist, the first few line of the file will appear.
-The status line at the bottom of your screen shows error messages and provides information and feedback, including the name of the file.

vi Modes

Command Mode
-Command mode is the mode you are in when you start (default mode)
-Command mode is the mode in which commands are given to move around in the file, to make changes, and to leave the file
-Commands are case sensitive: j not the same as J
-Most commands do not appear on the screen as you type them. Some commands will appear on the last line: : / ?
Insert (or Text) Mode
The mode in which text is created. (You must press at the end of each line unless you've set wrap margin.)
There is more than one way to get into insert mode but only one way to leave: return to command mode by pressing
When in doubt about which mode you are in, press
Basic cursor movement
k    Up one line
j    Down one line
h    Left one character
l    Right one character (or use )
w    Right one word
b    Left one word

Entering, Deleting, and Changing Text

From Command Mode
i    Enter text entry mode
x    Delete a character
dd    Delete a line
r    Replace a character
R    Overwrite text, press to end

    shell scripting

    Shell Scripting
    -> Shell is a UNIX Command Interpretor
    -> There are many versions of Shell but commone ones are: SH-> Shell CSH-> C Shell, BASH-> Born Again Shell, TSH-> Extended C Shell
    -> Shell Scripting is about writing a script which contains a list of commands that will be executed by Shell Interpretor.

    Writing Shell Script:

    -> We use editor like vi to write code
    -> Each Shell script start with the line: #!/bin/sh
    -> The Symbol # means the line is a comment but Symbol ! means next lines are are special and need be intepreted by shell.
    -> No Data types and variable declaration are used
    -> To display we use echo
    -> To access variable value we put dollar symbol($) in front of variable name
    -> After writing your script you change file permission to give execute previlege
    -> To run your script we use: ./FileName

    Example: Write a shell script that displays Hello IT Students

    Answer: #!/bin/sh
    hello="Hello IT Students"
    echo $hello
    echo "IT Students like LUNIX"
    Example2:

      • 1.LUNIX Operating System
      -> LUNIX is a command line operating system
      -> LUNIX is multi-user and multi-task operating system
      -> LUNIX was created in Bell Lab
      -> LUNIX has given birth of UNIX Like operating systems which are open source and support command line and Graphical User Intrface(GUI) interfaces such as FreeBSD, Linux, Ubuntu, Fedora, RedHat,...
      -> LUNIX has super user called root and is represented by /
      -> LUNIX like other operating system, manages hardware, applications and users
      -> LUNIX stores data in files and files are stored in directories
      -> LUNIX has a command interpretor called Shell

      Commands: ln, grep, head, tail, sort, chmod
      ->ln: is used to create a link to your file (like shortcut)
      => There are 2 kinds of link: soft link and hard link
      => hard link is created without option and a link can be accessed after original file was deleted
      => soft link is created using ln -s, content of a link is lost after original file is deleted
      Format: ln -s FileName LinkName
      ->grep is used to search in a file
      Format: grep "Word to search" FileName
      -> head: is used to display first lines. Example: head -5 MyFile.txt =>displays first 5 lines of text from MyFile.txt
      ->Tail: displays last lines. Example: tail -3 MyFile.txt=>displays last 3 lines from MyFile.txt
      ->sort: is used to sort the content. Example: cat MyFile.txt | sort -nr =>displays a reverse sorted list
      -> chmod: means change mode: used to change file permissions for users:
      In UNIX we have 3 kind of users: Owner, Group and Others and 3 access rights: read, write, execute.
      => There are 3 alternatives for each user: 111(or rwx)= 7=>means full access
      r-xr--r-x=545

      111=7=>rwx

      Introduction to vi editor
      ->An editor allows us to type text or program code and allows us to manipulate the content
      -> vi is a UNIX editor which is supported by all UNIX versions
      -> vi editor has 3 operating modes:
      1. Command mode
      2. Insert mode
      3. Last Line mode
      changing vi modes:
      -> by default after entering in vi editor we are in command mode
      To go to Insert mode from command mode we press i
      To go command mode from Insert mode we press esc key
      To go to last line from command mode we use colon(:)
      To go to last line mode from Insert mode, first we go to command mode then we press :
      Manipulating content
      >Moving mouse cursor:
      top-down: we use letter j
      bottom-up: we use letter k
      left-right: we use letter l
      right-left: we use letter h
      copy and paste:
      to copy one line we use: yy
      to copy 5 lines we use: 5yy
      to paste we use: p
      deleting in a file:
      to delete one character we use: x
      to delete a word we use: dw
      to delete a line we use: dd
      to delete 10 lines we use: 10dd
      NB: to manipulate the content of a file created using vi we must be in command mode.
      => to save and quit we use :wq
      =>to save and remain we use :w
      => to quit without saving we :q
